# demo_code

> Example application

## Build Setup

``` bash

# clone repository
git clone https://github.com/fc0de/demo_code.git

# create and activate virtual environment
python -m venv venv
source venv/bin/activate

# install dependencies
pip install -r requirements.txt

# run dev-server at 127.0.0.1:5000
flask --app app run

# open in browser
http://127.0.0.1:5000

```
